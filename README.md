Supernova catalog at: [https://sanjanagupta02.github.io/catalog.html](https://sanjanagupta02.github.io/catalog.html)

Data is taken from [Transient Name Server](https://www.wis-tns.org) (IAU) which is updated daily with new SNe and Open Supernova Catalog via [Aaron Geller's UniviewSNvTime repository](https://github.com/ageller/UniviewSNvTime/tree/master/rawdata/OpenSNCat.csv), which made historical data much easier to work with

Used custom scripts to convert the OSC csv data into the same JSON format as the TNS output and handle deduplication across the two sources, producing a single merged catalog.
